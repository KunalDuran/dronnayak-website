MAVLink v1 Frame Structure (8 - 263 bytes)
STX
1 byte
Start marker (0xFE). Marks the beginning of a MAVLink v1 message.
LEN
1 byte
Payload length (0-255). Number of bytes in the message data.
SEQ
1 byte
Sequence number (0-255). Counts each message sent.
SYS ID
1 byte
System ID. Identifies which device is sending.
COMP ID
1 byte
Component ID. Identifies which part of the system.
MSG ID
1 byte
Message ID (0-255). Defines message type.
PAYLOAD
0-255 bytes
The actual message data. Content varies by message type.
CRC
2 bytes
Checksum for error detection. Verifies message integrity.

MAVLink v2 Frame Structure (12 - 280 bytes)
STX
1 byte
Start marker (0xFD). Marks the beginning of a MAVLink v2 message.
LEN
1 byte
Payload length (0-255). Number of bytes in the message data.
INC
1 byte
Incompatibility flags. Features that v1 cannot understand.
COM
1 byte
Compatibility flags. Features that v1 can safely ignore.
SEQ
1 byte
Sequence number (0-255). Counts each message sent.
SYS ID
1 byte
System ID. Identifies which device is sending.
COMP ID
1 byte
Component ID. Identifies which part of the system.
MSG ID
3 bytes
Message ID (0-16M). Defines message type - now with extended range.
PAYLOAD
0-255 bytes
The actual message data. Content varies by message type.
CRC
2 bytes
Checksum for error detection. Verifies message integrity.
